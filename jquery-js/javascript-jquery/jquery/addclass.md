# jQ 增加CSS Classes

```javascript
// 針對多個element增加class
$("h1, h2, p").addClass("blue");

// 針對多個element移除class
$("h1, h2, p").removeClass("blue");

// 針對多個element切換class
$("h1, h2, p").toggleClass("blue");

// 針對一個element增加多個class
$("#div1").addClass("important blue");
```
